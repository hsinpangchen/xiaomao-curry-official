從 Java、Rails、iOS，到真的跑去賣咖哩飯。這不是勵志雞湯，比較像是一段工程師對工作、創業和現實的誠實筆記。

#品牌故事 #創業 #笑貓咖哩

iOS工程師賣咖哩飯之路

每個人都有不同的際遇，不同的際遇產生了不同的看法。可能我的軟體之路走的不是很順，從國防役開始用 Java 寫 web server、經歷 Ruby/Rails 的崛起，最後進入到 Mobile 的世界。

而 Rails 風行的時代也帶出了好多自動化想法、流程和工具。像是自動部署之類的，前端也從 jQuery 開始一直不斷地進化。在這段時間其實蠻興奮的，有點感受這是一波近代的軟體革命吧。

但同時因為技術大爆炸，所以要學的東西好多，每天幾乎看到深夜兩點，或眼睛張不開了才睡，似乎永遠追不完技術知識。那時候就開始感到焦慮，覺得自己怎樣都追不上西方世界的工程師，他們實在太屌了，可以創造出那些很棒的東西和想法。

加上自己對於軟體之外的商業世界也有想法，就第一次叛逃跑去開了超格林茶書店。

很多工程師都有一種夢，開個咖啡店，沒客人的時候寫 code，有客人的時候就做做飲料服務一下客人，一個非常好的理想。

但現實是，沒客人的時候，根本沒心情寫 code 啦。一直會想說，怎麼辦，是我哪裡做不好，都沒人耶，那房租怎麼辦？水電怎麼辦？是賣太貴嗎？會不會一直沒人撐下去。

有客人的時候，當然也得做飲料、洗東西、備料，只能說這樣的夢一整個是心理和體力上的折磨。

就在這樣獨立一人經營了一年七個月後，就決定停止這樣的生活。休息了幾個月後，就又回到軟體工程師的工作，投入了 Mobile 領域的 iOS 開發，一直到現在還持續著。

賣咖哩飯是最近這幾年，思考自己如果有天因為年紀變大，軟體又沒搞頭時，好像可以回屏東市老家賣咖哩飯，加上東港的海鮮，海鮮咖哩可能也有搞頭。

就這樣一個模糊的概念一直放在我心中，遇到一些相關的咖哩食譜、書籍、想法，我就都會收集起來。去年在北海道，也因此吃了不少咖哩，日式的、尼泊爾的、印度咖哩、泰式綠椰汁咖哩。

真正讓賣咖哩飯這個點子實現的契機，可能還是留在後面再說。這個契機出現後，剛好在新莊住家樓下的路上，有一個攤位在出租，算一算店租、餐車等器具，還算是一筆負擔得起的錢，比 MacBook Pro 還便宜，XD。

我跟老婆就衝了，展開了我們的攤販咖哩人生。

待續。
